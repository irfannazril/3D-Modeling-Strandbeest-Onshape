# 3D Mechanical Design: Strandbeest

## 📖 Deskripsi Proyek
Proyek ini merupakan eksplorasi desain mekanikal digital untuk memodelkan robot Strandbeest. Fokus utama dari proyek ini adalah mempelajari teknik *3D modeling* dan merancang mekanisme *Jansen Linkage* secara virtual menggunakan platform Onshape.

## 🎯 Tujuan Pembelajaran (Project Scope)
Proyek ini difokuskan murni pada tahap desain (*Proof of Concept*), dengan target pencapaian:
* Menguasai antarmuka dan *tools* pada platform Onshape.
* Menerapkan prinsip desain parametrik untuk membuat komponen yang presisi.
* Mensimulasikan pergerakan mekanis dari *Jansen Linkage* untuk memastikan setiap sendi/engsel dapat berputar tanpa benturan fisik (*collision*).

## 🛠️ Tools yang Digunakan
* **Software CAD:** Onshape (Cloud-based 3D CAD)
* **Format Output:** File `.stl` / `.step` (Siap untuk *3D Printing*)

## 💡 Inpirasi dan Sumber Blueprint
Proyek ini terinspirasi oleh patung kinetik raksasa yang dirancang agar dapat berjalan sendiri menggunakan tenaga angin karya seniman sekaligus fisikawan asal Belanda, Theo Jansen. Blueprint project ini saya ambil dari situs blog NumaVIG (https://numavig.com/plans/Spider-Robot)
